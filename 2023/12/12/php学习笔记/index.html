php

  • PHP（全称：PHP：Hypertext Preprocessor，即”PHP：超文本预处理器”）是一种通用开源脚本语言。

  • PHP 脚本在服务器上执行。

PHP 是一门弱类型语言

编写代码时不必向 PHP 声明该变量的数据类型。

PHP 会根据变量的值，自动把变量转换为正确的数据类型。

在强类型的编程语言中，必须在使用变量前先声明（定义）变量的类型和名称。

PHP echo 和 print 语句

echo 和 print 区别:

  • echo - 可以输出一个或多个字符串
  • print - 只允许输出一个字符串，返回值总为 1

提示：echo 输出的速度比 print 快， echo 没有返回值，print有返回值1。

php数据类型

PHP 变量存储不同的类型的数据，不同的数据类型可以做不一样的事情。

PHP 支持以下几种数据类型:

  • String（字符串）

  • Integer（整型）

    PHP [var_dump() ]函数返回变量的数据类型和值：

    Float（浮点型）

    Boolean（布尔型）

    Array（数组）

    Object（对象）

    NULL（空值）

    NULL 值表示变量没有值。NULL 是数据类型为 NULL 的值。

    NULL 值指明一个变量是否为空值。 Resource（资源类型）
