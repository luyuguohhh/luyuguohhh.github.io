二.变量

变量:可以发生改变的一个量，变量是用来区分不同数据的，可以指向一个内存空间，帮我们存储一些数据

变量的命名规范:
1、必须是数字或字母或下换线组成，
2、不能是数字开头，更不能是纯数字
3、不能用python的关键字(if /for /while /break /true)
4、不要用中文
5、不要太长
6、要有意义
7、推荐使用下换线命名或者驼峰命名（首字母大写）

三.数据类型

1、数字: + - * /
整数，int—-加减乘除，大小比较

/ (常规除)
如：

5 / 2 = 2.5

解释：平常除法是什么结果就是什么结果。

//（地板除）
如:

5 // 2 = 2 （5 ÷ 2 = 2.5）

5 // 3 = 1 (5 ÷ 3 = 1.6666666666666667)

解释：地板除，只去除完之后的整数部分。

% （取余数）

如：

5 % 2 = 1 （5 - 2*2 = 1）

4 % 2 = 0 （4 - 2*2 = 0）

解释： 就是一个取余数的操作，除开被除数的倍数，余下几就得几。上面红色的数字带表被除数的倍数。

小数（浮点数）：float

八、字典

1、定义

首先，字典是以键值对的形式进行存储数据的
字典的表示方式: {key:value, key2:value, key3:value}

字典的key必须是可哈希的数据类型
字典的value可以是任何数据类型

不可哈希: python中的set集合进行数据存储的时候，需要对数据进行哈希计算，根据计算出来的哈希值进行存储数据
set集合要求存储的数据必须是可以进行哈希计算的。
可变的数据类型，list, dict, set

可哈希:不可变的数据类型，int, str, tuple, bool.
